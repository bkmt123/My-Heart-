Airtable Setup

Connect to Airtable so both of you can access the dashboard from any device — phone, tablet, or computer.

One-time setup (5 minutes):
1. Go to airtable.com → create a free account
2. Create a new Base (name it anything, e.g. "Our Dashboard")
3. Inside it, create a Table with two fields:
  • DashboardKey (Single line text)
  • StateJSON (Long text)
4. Go to airtable.com/create/tokens → create a token with data.records:read and data.records:write scopes for your base
5. Copy the Base ID from the URL: airtable.com/appXXXXXX/...
6. Paste everything below and hit Save.
